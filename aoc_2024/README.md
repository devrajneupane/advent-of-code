# Advent of Code 2024

Advent of Code 2024 solutions, written in Rust

## Usage

1. Run test for specific mod

```bash
cargo test <MOD_NAME>

# Eg:  cargo test day01
```

2. Run solution

Put your input in `input.txt` file and run following command

```bash
cargo run <DAY> <PART>

# Eg:
# cargo run 01 1
# cargo run 01 2
```
